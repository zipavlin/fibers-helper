# Short Introduction

This package is in public alpha.

Fibers Helper is a developer oriented tool - a collection of assorted helpers that are used in other parts of Fibers, primarily for collection of details and modification of different Laravel part.

# Quick Start

# install composer package
composer require-dev fibers/helper

# import facades
use Fibers/Helper/Facades/ModelHelper;

# use helper
$modelInfo = ModelHelper::fromClass('App\Models\Place')->collect('attributes', 'relationships');

# Helpers

Memory Helper - build, store and run an in-memory queue of commands
Model Helper - get info about model, it's attributes, relationships, ...
Models Helper - collect info about collection of models
Template Helper - template writer
User Helper - collect info about user model and auth type
View Helper - collect info about views
Class Helper - collect class info and modify class
